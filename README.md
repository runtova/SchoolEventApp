# School Event Management System

Система учёта проведения школьных мероприятий

## 📋 Описание проекта

Desktop-приложение для управления школьными мероприятиями, участниками и организаторами.
Разработано на платформе .NET 10 с использованием WPF и Microsoft SQL Server.

## 🚀 Функциональные возможности

### Для администратора
- Полный CRUD организаторов, участников, мероприятий
- Формирование отчётов (Excel, PDF, Word)
- Резервное копирование и восстановление БД
- Дашборд с аналитикой

### Для организатора
- Управление только своими мероприятиями
- Просмотр участников
- Экспорт списков участников

### Для участника
- Просмотр доступных мероприятий в виде карточек
- Запись / отмена записи на мероприятия
- Личный кабинет с историей записей

## 🛠 Технологии

- **.NET 10** (Windows Desktop)
- **WPF** (Windows Presentation Foundation)
- **Microsoft SQL Server**
- **FastReport.OpenSource** (отчёты PDF)
- **ClosedXML** (экспорт в Excel)

## 📁 Структура БД

- `Organizer` — организаторы мероприятий
- `Participant` — участники (ученики)
- `Eventt` — мероприятия
- `Participation` — связи участников с мероприятиями
- `Users` — пользователи системы (авторизация)

2. Разверните базу данных School_event (выполните SQL-скрипты из папки `SQL/`)
3. Настройте строку подключения в `App.config`
4. Запустите `SchoolEventApp.exe`
